Billplz payment integration for FluentCart.

== Description ==

Billplz payment integration for FluentCart.

== How it works ==

This plugin will connect to [Billplz API](https://www.billplz.com/api) to accept payments via multiple payment methods such as FPX online banking, credit card, Paypal and eWallet.

[Sign up as a merchant](https://www.billplz.com/) to get started.

You may use our sandbox environment for testing: [Billplz Sandbox](https://www.billplz-sandbox.com/)

== Installation ==

= Minimum Requirements =

* WordPress 6.8 or later
* FluentCart

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of Billplz for FluentCart, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “Billplz for FluentCart,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the Billplz for FluentCart plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
