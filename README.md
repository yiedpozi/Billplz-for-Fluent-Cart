# Billplz for FluentCart

Accept payment using Billplz by using this plugin.

## Requirements

* PHP 7.0 or later
* WordPress 6.8 or later
* FluentCart

## Installation

1. Download the latest plugin file: [billplz-for-fluent-cart.zip](https://github.com/Billplz/Billplz-for-Fluent-Cart/releases)
2. Log in to **WordPress Dashboard**.
3. Navigate to **Plugins** > **Add Plugin** > **Upload Plugin**.
4. Upload the plugin file and click **Activate**.

## Configuration

1. Log in to your **WordPress Dashboard**.
2. Navigate to **Plugins** > **Installed Plugins**.
3. Find **Billplz for Fluent Cart** in the plugins list and click **Settings** to configure.
4. Enter your API credentials in the payment settings.
   - You can obtain your API credentials from your [Billplz dashboard](https://dashboard.billplz.com/enterprise/setting).
5. Enable **Payment Activation** to activate Billplz as a payment method.
6. Click **Save Settings**.
